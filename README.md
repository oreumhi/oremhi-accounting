# 주식회사 오름히 - 회계관리 시스템 v4.0

광고대행사 전용 회계관리 웹앱. 핸드폰/노트북/PC 어디서든 접속 가능.

## 전체 기능 (16개 메뉴 + 15개 추가기능)

### 메뉴
| 메뉴 | 설명 |
|------|------|
| 대시보드 | 매출/지출/순이익 + **세금마감알림** + **예산초과경고** |
| 매출관리 | 매출 기록 + **엑셀 내보내기** |
| 지출관리 | 카드/현금/간편결제 + **즐겨찾기** + **중복감지** + **영수증이미지보기** |
| 정기지출 | 구독/임대/보험 반복비용 |
| 통장관리 | 입출금/잔액 관리 |
| 거래처 | 미수금/미지급금 + **거래처별 수익률** |
| 세금계산서 | 발행/수취, 부가세 자동계산 |
| 부가세 | 분기별 자동 집계 |
| 급여/원천세 | 4대보험/소득세 자동계산 |
| 예산관리 | **카테고리별 예산 설정 + 초과알림** |
| 리포트/분석 | **월별비교** + **연도별리포트** + **경비비율분석** |
| 계약/견적 | 계약서/견적서 관리 |
| 증빙관리 | 증빙자료 보관현황 |
| 영수증스캔 | AI 인식 + **이미지 영구저장** |
| 메모장 | **메모/할일 기록** |
| 설정 | **비밀번호잠금** + **글자크기조절** + **백업/복원** |

- ⏰ 세금 신고 마감일 알림
- 💾 데이터 백업/복원 (JSON + 엑셀)
- 📌 메모장
- 📊 월별 비교 분석
- 💰 거래처별 수익률
- 🎯 예산 설정/초과 알림
- ⭐ 자주 쓰는 거래 즐겨찾기
- ⚠️ 중복 입력 방지

## 배포 방법

### 1단계: GitHub에 코드 올리기
1. github.com 가입
2. 오른쪽 위 `+` → `New repository` → 이름: `oremhi-accounting` → Create
3. `uploading an existing file` 클릭
4. 이 폴더의 모든 파일/폴더를 드래그 앤 드롭
5. `Commit changes` 클릭

### 2단계: Supabase 데이터베이스
1. supabase.com → GitHub로 로그인
2. `New Project` → 이름: `oremhi-accounting` → Region: Tokyo → Create
3. 왼쪽 `SQL Editor` → `supabase-setup.sql` 내용 전체 붙여넣기 → Run
4. `Settings` → `API` 에서:
   - **Project URL** 복사
   - **anon public** 키 복사

### 3단계: Vercel 배포
1. vercel.com → GitHub로 로그인
2. `Add New` → `Project` → `oremhi-accounting` Import
3. Environment Variables에 추가:
   - `VITE_SUPABASE_URL` = Project URL
   - `VITE_SUPABASE_ANON_KEY` = anon public 키
   - `ANTHROPIC_API_KEY` = (선택) 영수증 AI스캔용
4. `Deploy` 클릭 → 완료!

### 핸드폰 홈화면 추가
- 안드로이드: Chrome → 메뉴(⋮) → 홈 화면에 추가
- 아이폰: Safari → 공유 → 홈 화면에 추가
